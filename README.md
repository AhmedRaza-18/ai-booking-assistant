# AI Booking Assistant

AI-powered voice and chat agent that captures leads, qualifies them, and books appointments automatically.

## Features
- 24/7 AI voice agent (phone calls)
- Text chat support
- Lead qualification (budget, timeline, service)
- Google Calendar integration
- SMS confirmations
- Voice call reminders

## Tech Stack
- **Backend:** FastAPI (Python)
- **AI:** OpenAI GPT / Local LLM
- **Voice:** Twilio
- **Calendar:** Google Calendar API
- **Database:** SQLite
- **Hosting:** Railway / Render
